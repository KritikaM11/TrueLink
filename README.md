# TrueLink 🔗

TrueLink is a real-time video conferencing and messaging application built with the MERN stack and WebRTC. It allows users to create secure meeting rooms, chat in real-time, and share their screens seamlessly.

## ✨ Features
* **Video & Audio Calling:** High-quality peer-to-peer communication powered by WebRTC.
* **Real-Time Messaging:** Live, synchronized chat inside meeting rooms using Socket.io.
* **Screen Sharing:** Share your screen with other participants with a single click.
* **Guest Mode & Authentication:** Join quickly as a guest via a meeting code, or log in to keep a history of your past meetings.
* **Smart Layouts:** The UI automatically adjusts for 1-on-1 calls, multi-user grid views, or pinned speakers.

## 🛠️ Tech Stack
* **Frontend:** React, React Router, standard CSS
* **Backend:** Node.js, Express.js
* **Real-Time & Media:** Socket.io, WebRTC
* **Database:** MongoDB, Mongoose
